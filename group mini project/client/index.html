This is a web application that:

a) has a user login page with a forgot password link that sends a reset password message to the user email.
b) allows user to login after entering correct username and password
c) Presents a form to the user to enter the following contact detail
i) mobile phone number
ii) email address
iii) registration number
d) search for contacts of person using their registration number

This is a website that authenticates users by enabling them to register with their details.

Then login to the website after they login they will be displayed with a contact details which they will fill in their contact details and everything will be saved on database
